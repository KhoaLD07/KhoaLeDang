WINDOWS SERVICES AND SERVICE ACCOUNTS

TOPIC: CREATING, PROTECTING AND CONFIGURING WINDOWS SERVICES

1. Notes and Explanation

Local System: Highest Privilege, it has access to all files, registry and system resources. Windows Defender and Windows Update will use this account

Network Service: This account has low priv (about the same as a normal user) on the computer. But, it's allowed to the Internet with credentials of the computer. DNS client or DHCP services needs internet. But don't need to edit or delete the system.

Local Services: This account has low privlege, it doesn't have access to the internet (if they do it will be anonymous, no credentials). Windows Audio, Print Spooler.

Virtual Account: Microsoft create this type to isolate. Instead of using lots of Local Service, Windows create a virtual account for each service. THis account doesn't need password. When you use SQL server, system will run it as NTSERVICE account. If this account got hacked, the hacker will only have access to the data of SQL but not others services.

The differents between Local System, Network Service and Local Service. Local System will have all access, Network System is allowed to connect to the Internet with the computer credentials and have access to the LAN folder. Local Service will still have access but it can't access to the LAN folder because of the anonymous credentials access denied. Both Network service and Local service don't have the privileges to Write/Edit/Delete inside the C:\Windows folder.

  • This is the SDDL string (Security Descriptor Definition Language):
  • "SY" means SYSTEM account:
    • Has full control (Start, Stop, Delete, Config)
  • "BA" means Built-in Administrators:
    • Has ONLY Read access (Cannot Stop, Cannot Delete)
  • D:(A;;CCLCSWRPWPDTLOCRRC;;;SY) — System rule
  • (A;;CCLCSWLOCRRCWD;;;BA) — Built-in Administrator
  • Explanation of SDDL format:
    • D: declare DACL, tells Windows this is an SDDL list
    • A (Allow): self-explanatory
    • SY (System): SYSTEM account is granted the following permissions:
  • CCLCSWRPWPDTLOCRRC (permission codes):
    • CC (Service_Query_Config): see the settings of the service (e.g., backing .exe)
    • LC (Service_Query_Status): see the status (e.g., running, stopped, etc.)
    • SW (Service_Enumerate_Dependents): see the list of services this service depends on
    • RP (Service_Start): allowed to start the service
    • WP (Service_Stop): allowed to stop the service
    • DT (Service_Pause_Continue): allowed to pause or resume the service
    • LO (Service_Interrogate): allowed to send a signal requesting a status report
    • CR (Service_User_Defined_Control): allowed to send custom control codes (e.g., in service_ctrl_handler)
    • RC (Read_Control): see Discretionary Access Control List (DACL)
      • Example: When an account (admin) tries to stop the service, Windows looks up "allowed ADMIN to STOP" in the DACL. If not included, Windows throws "ERROR_ACCESS_DENIED".
    • WD (Write DAC): allowed to unprotect. WARNING: Without this privilege, you can't unprotect the service.
    • SD (DELETE): allowed to delete the service
  • Difference between BA and SY:
    • BA does not have RP (can't start), WP (can't stop), DT (can't pause).
  • Functions used:
    • ConvertStringSecurityDescriptorToSecurityDescriptorW: Converts the SDDL string into a SECURITY_DESCRIPTOR struct.
    • SetServiceObjectSecurity: Applies the SECURITY_DESCRIPTOR to the service.
  • To unprotect the service:
    • Change the SDDL string into D:(A;;CCLCSWRPWPDTLOCRRCWDSD;;;BA) to give full access back to administrators.

To create recover actions we need a SERVICE_FAILURE_ACTIONSW struct to contains all the SC_ACTION config. then we begin to config by doing action[0].Type for the type of action and delay for the time. 

typedef struct _SC_ACTION {
  SC_ACTION_TYPE Type;
  DWORD          Delay;
} SC_ACTION, *LPSC_ACTION;

After that we put all our actions into the SERVICE_FAILURE_ACTIONSW struct. 

typedef struct _SERVICE_FAILURE_ACTIONSW {
   DWORD dwResetPeriod; // The time, in seconds, after which to reset the failure count to zero if there are no failures.
   LPWSTR lpRebootMsg; // contains the message to be broadcast to server users before rebooting. It's needed when we use SC_ACTION_REBOOT.
   LPWSTR lpCommand; // contains the command line of the process for the process creation function to execute. It's needed when we use SC_ACTION_RUN_COMMAND.
   DWORD cActions; // number of Elements in the SC_ACTION array.
   SC_ACTION* lpsaActions; // a pointer to the SC_ACTION array which have all of our ACTIONS.
 } SERVICE_FAILURE_ACTIONSW,
  *LPSERVICE_FAILURE_ACTIONSW;

// Stop and delete a service by name
void stop_and_delete(LPCWSTR service_name) {
    SC_HANDLE service_handle = NULL;
    SC_HANDLE service_manager_handle = NULL;
    BOOL success = FALSE;
    SERVICE_STATUS status = {};
    service_manager_handle = OpenSCManagerW(NULL, SERVICES_ACTIVE_DATABASEW, SC_MANAGER_CONNECT);
    service_handle = OpenServiceW(service_manager_handle, service_name, SERVICE_STOP | DELETE);
    if (!service_handle) {
        printf("\nFAILED TO OPEN SERVICE: %lu", GetLastError());
        CloseServiceHandle(service_manager_handle);
        return;
    }

    success = ControlService(service_handle, SERVICE_CONTROL_STOP, &status);
    if (!success) {
        printf("\nNote: Service might already be stopped (%lu). Proceeding to delete...", GetLastError());
    }
    success = DeleteService(service_handle);
    if (!success) {
        printf("\nFAILED TO DELETE SERVICE: %lu", GetLastError());
        CloseServiceHandle(service_handle);
        CloseServiceHandle(service_manager_handle);
        return;
    }
    printf("\nSERVICE %ws deleted!", service_name);
    CloseServiceHandle(service_handle);
    CloseServiceHandle(service_manager_handle);
}
// Protect a service so only SYSTEM can modify/delete it
void protect_service(LPCWSTR service_name) {
    SC_HANDLE service_handle = NULL;
    SC_HANDLE service_manager_handle = NULL;
    service_manager_handle = OpenSCManagerW(NULL, SERVICES_ACTIVE_DATABASEW, SC_MANAGER_CONNECT);
    if (!service_manager_handle) {
        printf("\nFAILED TO OPEN SCMANAGER: %lu", GetLastError());
        return;
    }
    service_handle = OpenServiceW(service_manager_handle, service_name, WRITE_DAC | READ_CONTROL);
    if (!service_handle) {
        CloseServiceHandle(service_manager_handle);
        printf("\nFAILED TO OPEN SERVICE FOR PROTECTING: %lu", GetLastError());
        return;
    }
    LPCWSTR sddl = L"D:(A;;CCLCSWRPWPDTLOCRRC;;;SY)(A;;CCLCSWLOCRRCWD;;;BA)"; // SYSTEM: Full, Admins: Read only
    PSECURITY_DESCRIPTOR pSD = NULL;
    if (ConvertStringSecurityDescriptorToSecurityDescriptorW(sddl, SDDL_REVISION_1, &pSD, NULL)) {
        if (SetServiceObjectSecurity(service_handle, DACL_SECURITY_INFORMATION, pSD)) {
            printf("\nSERVICE %ws IS NOW PROTECTED!", service_name);
        } else {
            printf("\nFAILED TO PROTECT SERVICE: %lu", GetLastError());
        }
        LocalFree(pSD);
    } else {
        printf("\nFAILED TO PARSE SECURITY STRING: %lu", GetLastError());
    }
    CloseServiceHandle(service_handle);
    CloseServiceHandle(service_manager_handle);
}

// Set recover actions and required privileges for a service
void set_recover_action(LPWSTR service_name) {
    SC_HANDLE service_manager_handle = NULL;
    SC_HANDLE service_handle = NULL;
    BOOL success = FALSE;
    SERVICE_FAILURE_ACTIONSW failure_actions = { 0 };
    service_manager_handle = OpenSCManagerW(NULL, SERVICES_ACTIVE_DATABASEW, SC_MANAGER_CONNECT);
    if (!service_manager_handle) {
        printf("\nFAILED TO OPEN SCMANAGER TO SET RECOVER ACTION");
        return;
    }
    service_handle = OpenServiceW(service_manager_handle, service_name, SERVICE_START | SERVICE_CHANGE_CONFIG);
    if (!service_handle) {
        CloseServiceHandle(service_manager_handle);
        printf("\nFAILED TO OPEN SERVICE TO SET RECOVER ACTION");
        return;
    }
    SC_ACTION action[3];
    action[0].Type = SC_ACTION_RESTART;
    action[0].Delay = 5000;
    action[1].Type = SC_ACTION_RESTART;
    action[1].Delay = 10000;
    action[2].Type = SC_ACTION_NONE;
    action[2].Delay = 0;
    failure_actions.dwResetPeriod = 86400;
    failure_actions.lpRebootMsg = NULL;
    failure_actions.lpCommand = NULL;
    failure_actions.cActions = 3;
    failure_actions.lpsaActions = action;
    LPCWSTR privileges = L"SeCreateGlobalPrivilege\0SeImpersonatePrivilege\0";
    SERVICE_REQUIRED_PRIVILEGES_INFO info;
    info.pmszRequiredPrivileges = (LPWSTR)privileges;
    if (ChangeServiceConfig2(
            service_handle,
            SERVICE_CONFIG_REQUIRED_PRIVILEGES_INFO,
            &info)) {
        printf("\nadded required privileges");
    }
    success = ChangeServiceConfig2W(service_handle, SERVICE_CONFIG_FAILURE_ACTIONS, &failure_actions);
    if (success == TRUE) {
        printf("\nRECOVER ACTIONS CONFIGURED!");
    } else {
        printf("\nFAILED TO CONFIGURED ACTION: %lu", GetLastError());
    }
    CloseServiceHandle(service_handle);
    CloseServiceHandle(service_manager_handle);
}

#define SERVICE_NAME "MyNoteService"
VOID WINAPI ServiceCtrlHandler(DWORD ctrl_code) {
    switch (ctrl_code) {
    case SERVICE_CONTROL_STOP:
        gservice_status.dwCurrentState = SERVICE_STOP_PENDING;
        SetServiceStatus(gstatus_handle, &gservice_status);
        SetEvent(gservice_stop_event);
        break;
    default:
        break;
    }
}
DWORD WINAPI ServiceWorkerThread(LPVOID lp_param) {
    while (WaitForSingleObject(gservice_stop_event, 0) != WAIT_OBJECT_0) {
        show_message();
        Sleep(3000);
    }
    CloseHandle(gservice_stop_event);
    gservice_status.dwCurrentState = SERVICE_STOPPED;
    SetServiceStatus(gstatus_handle, &gservice_status);
    return ERROR_SUCCESS;
}

extern "C" __declspec(dllexport) VOID WINAPI ServiceMain(DWORD argc, LPTSTR* argv) {
    gstatus_handle = RegisterServiceCtrlHandlerA(SERVICE_NAME, ServiceCtrlHandler);
    if (!gstatus_handle) return;
    gservice_status.dwServiceType = SERVICE_WIN32_SHARE_PROCESS; // IMPORTANT NEED TO RUN THE SAME PROCESS
    gservice_status.dwControlsAccepted = SERVICE_ACCEPT_STOP;
    gservice_status.dwCurrentState = SERVICE_START_PENDING;
    gservice_status.dwWin32ExitCode = 0;
    gservice_status.dwServiceSpecificExitCode = 0;
    gservice_status.dwCheckPoint = 0;
    gservice_status.dwWaitHint = 0;
    SetServiceStatus(gstatus_handle, &gservice_status);
    gservice_stop_event = CreateEvent(NULL, TRUE, FALSE, NULL);
    if (gservice_stop_event == NULL) {
        gservice_status.dwCurrentState = SERVICE_STOPPED;
        gservice_status.dwWin32ExitCode = GetLastError();
        SetServiceStatus(gstatus_handle, &gservice_status);
        return;
    }
    gservice_status.dwCurrentState = SERVICE_RUNNING;
    SetServiceStatus(gstatus_handle, &gservice_status);

    HANDLE hThread = CreateThread(NULL, 0, ServiceWorkerThread, NULL, 0, NULL);
    if (hThread) {
        CloseHandle(hThread);
    }
}
